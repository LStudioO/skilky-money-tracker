# F19: Export

**Priority:** Post-MVP
**Depends on:** F05 (Expense CRUD)

## Summary

Export expense data as CSV or PDF for personal records, tax purposes, or migration.

## Key Features

- Export as CSV (machine-readable)
- Export as PDF (printable report)
- Filter by date range
- Include/exclude categories
- Download from app or email

## CSV Format

```csv
Date,Name,Amount,Currency,Category,Note,Input Type
2026-03-21,Milk,45.00,UAH,Food,,TEXT
2026-03-21,Taxi,120.00,UAH,Transport,,AUDIO
```
