### Shared Validation

Value classes with `require()` blocks enforce domain rules at construction. Shared between client and server — validated in `:shared:models`, so rules never drift.

```
value class Email(val value: String) {
    init { require(value.matches(EMAIL_REGEX) && value.length <= 254) }
}
```

| Field | Rule |
|-------|------|
| Email | Valid format (RFC 5322 simplified), max 254 chars |
| Password | Min 8 chars, at least 1 letter + 1 digit |
| Display name | 1–100 chars, trimmed |
| Expense amount | > 0, max 12 digits |
| Category name | 1–100 chars |
| Category color | Valid hex (#RRGGBB) |
